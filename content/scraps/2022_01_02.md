# 開発元を検証できないツールの実行
安全なのはわかってるけど、開発元が検証できずに、Macで拒否されるときの対処法

以下のコマンドで解除できる。
```shell
$ sudo spctl --master-disable
```
戻すときはこうする。
```shell
$ sudo spctl --master-enable
```
